## Template Notes (Meta — Don't Include in Actual Guides)

**Section ordering principle:** Setup → Body position (bottom-up) → Movement (eccentric → concentric) → Diagnostics (what you should feel, checklist, mistakes, red flags) → Programming → Mastery (advanced, plain English).

**What changes per lift type:**
- **Bodyweight lifts (pull-ups, dips, pistols):** Progression Path is mandatory.
- **Big-3 (squat, deadlift, bench):** Recovery Notes section is mandatory.
- **Pure strength lifts (deadlift, squat):** keep tiered cues AND "What You Should Feel" — beginners need both, even on power lifts.
- **Isolation/hypertrophy lifts (curls, lateral raises):** brace section can shrink to one line.
- **Lifts with no grip element:** drop Step 4.

**Cue discipline:**
- One cue per set. Stacking cues = paralysis.
- Beginner cues lean **internal + analogy** (mind-muscle, "imagine X")
- Advanced cues lean **external + terse** (action on environment, "push the floor away")
- Intermediate is the bridge — outcome-based ("stretch the chest, not the shoulders")

**Severity labeling:**
- Default new mistakes to "Form-only" until proven otherwise
- "Injury-risk" should be reserved for things that genuinely hurt people (rounded back deadlifts, knee cave under heavy load, dropped lockouts on OHP)

**Length target:** 200–500 lines per guide. Longer = beginner overwhelm. Shorter = missing the failure modes that matter.

**Tone rules:**
- Imperatives, not suggestions
- One cue per bullet — never stack two
- Failure modes inline ("too steep → shoulder press") so they learn the *why*
- Tables for comparisons (mistakes, progression stages)
- Plain English block at the end, never the start
- *Italic* = cues only. **Bold** = key terms / nouns. Don't mix the two.
